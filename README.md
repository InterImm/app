# 火星时钟

[下载](https://github.com/InterImm/app/releases)

## 开发

本地测试：

```bash
# Clone this repository
git clone --recursive https://github.com/InterImm/app.git
# Go into the repository
cd app/cal/
# Install dependencies
npm install
# Run the app
npm start
```

发布(Mac)：

```bash
# Clone this repository
git clone --recursive https://github.com/InterImm/app.git
# Go into the repository
cd app/cal/
# Install dependencies
npm install
# Modify
# npm build app 
npm run build
```
