# 星际移民中心的应用程序


## 火星时钟

实时显示火星当前的时刻和日期，并且可以输入地球时间查看对应的火星时间。火星历法采用 [星际移民之书这里的定义](http://book.interimm.org/navigator/mars.html#marscalendar)。
